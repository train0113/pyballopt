사용자 입력 안내
- 공 추가:
	마우스 왼쪽 버튼 클릭 → 마우스 위치에 공 5개 추가.
- 공 제거:
	Shift + Backspace → 화면의 모든 공 제거.
	Backspace + 왼쪽 클릭 → 클릭한 위치 주변 반경 내 공 제거.
- 공 크기 조절:
	Q 키 → 공 크기 줄이기 (최소 5픽셀).
	W 키 → 공 크기 늘리기.
청크 시각화:
	E 키 → 청크 그리드 및 점유 상태 표시.
공 움직임 제어:
	오른쪽 클릭 → 마우스 방향으로 공 끌어오기.
	Shift + 오른쪽 클릭 → 마우스 방향으로 공 밀기.
